# Travelers Info Junction 🌍

A modern travel information platform that helps users explore destinations, discover states and attractions, plan trips, and access useful travel insights through an intuitive and responsive web interface.

## Features

* Explore popular travel destinations
* State-wise travel information
* Interactive trip planning
* Responsive and user-friendly design
* Fast and lightweight frontend
* Backend API support for dynamic features
* Clean and modern UI

## Installation

### Clone Repository

```bash
git clone https://github.com/solankybhavana2-lgtm/travelers-info-junction.git
```

### Frontend

Open the `tij-frontend` folder and run:

```bash
index.html
```

### Backend

Navigate to backend folder:

```bash
cd tij-backend
npm install
npm start
```
